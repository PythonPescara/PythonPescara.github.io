Codice di Condotta

Le regole di comportamento della community Python Pescara.

Chi siamo

Python Pescara è una comunità locale auto-organizzata di volontari appassionati di Python che si incontra a Pescara e nei dintorni.

Facciamo parte della rete delle comunità italiane di Python Italia, un’associazione di promozione sociale formalmente costituita, alla quale è possibile associarsi e che ha un proprio Codice di Condotta:

https://www.python.it/comunita/

Il nostro impegno

Vogliamo che Python Pescara sia un luogo sicuro, aperto e accogliente per tutte le persone, senza distinzione di genere, origine o altre caratteristiche personali, e chiediamo a chi partecipa di contribuire con rispetto, ascolto e collaborazione.

Non sono accettati comportamenti di molestia o discriminazione, linguaggio offensivo o umiliante, intimidazioni o minacce, contatti fisici non appropriati o condivisione non richiesta di contenuti sessuali o violenti.

Ambito di applicazione

Questo Codice di Condotta si applica agli incontri in presenza organizzati da Python Pescara e ai canali di comunicazione online ufficiali del gruppo, il cui elenco è disponibile sul sito: https://pescara.python.it

Segnalazioni

Se osservi un comportamento che viola questo Codice di Condotta, puoi scrivere a: pescara@python.it

Le segnalazioni saranno gestite in modo riservato dagli amministratori di Python Pescara, nel rispetto della privacy delle persone coinvolte.

Applicazione

Gli organizzatori possono, in base alla gravità dei fatti, richiamare una persona, chiedere di interrompere un comportamento, impedire la partecipazione a un incontro in presenza o rimuovere una persona dai canali online del gruppo. Le decisioni vengono prese caso per caso con attenzione e buonsenso.

Python Pescara non è un ente formale e non assume obblighi legali verso i partecipanti, ma si impegna a prendere seriamente ogni segnalazione e a intervenire per mantenere un ambiente rispettoso e sicuro.

Relazione con altri Codici di Condotta

Quando Python Pescara partecipa o collabora ad eventi organizzati da Python Italia, o iniziative della Python Software Foundation, si applica il Codice di Condotta dell’organizzazione ospitante.

Questo documento si applica specificamente alle attività proprie di Python Pescara.
